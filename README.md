Tailwind CSS has been integrated into this project. To build the compiled CSS, follow these steps:

1. Install Node.js (v16+ recommended) and npm.
2. From the project root run:

   npm install
   npm run build:css

To run Tailwind in watch mode during development:

   npm run watch:css

The compiled CSS will be written to `static/assets/css/tailwind.css` and is already included in the base template.
